🍞 Toasts & Snackbars

Interactive notification components for your web applications

🍳 Toasts 🍞

Quick, non-intrusive notifications that typically appear in the top-right corner.
Perfect for:

  • ✨ Success confirmations
  • ℹ️ Information updates
  • ⚠️ Warning messages
  • ❌ Error notifications

🍽️ Snackbars 🍫

More prominent notifications that appear at the bottom of the screen. Great for:

  • 🎯 Important actions
  • 🔄 Status updates
  • 📱 Mobile-friendly alerts
  • 👆 User interactions

⌨️ Keyboard Shortcuts

Alt + T Show random toast
Alt + S Show random snackbar
Alt + C Clear all notifications
