📖 Getting Started

🚀 Quick Start

  1. Create slides - Click any of the slide type buttons above
  2. Edit slides - Hover over a slide and click "Edit"
  3. Reorder slides - Drag and drop slides to reorder them
  4. Present - Click the "Present" button to start your slideshow
  5. Save - Save your presentation to local storage

🎯 Key Features

  • Responsive Design - Works on all devices
  • Multiple Slide Types - Title, text, image, question, and code slides
  • Auto-play - Automatic progression through slides
  • Keyboard Navigation - Full keyboard support
  • Export Options - Save and load presentations

📝 Slide Types

📄 Title Slide

Perfect for introduction slides, section dividers, or title pages. Includes main title and subtitle fields.

📝 Text Slide

For content with bullet points or paragraphs. Great for lists, explanations, and detailed information.

🖼️ Image Slide

Showcase images with optional captions. Supports any web-accessible image URL.

❓ Question Slide

Interactive Q&A slides with multiple choice options. Perfect for audience engagement.

💻 Code Slide

Display code snippets with syntax highlighting. Specify the programming language for better formatting.

💡 Pro Tips

🎨 Design Tips

  • Keep text concise - aim for 6-7 words per line
  • Use high-contrast images for better readability
  • Limit bullet points to 3-5 per slide
  • Choose readable fonts and appropriate sizes

🚀 Performance

  • Optimize images before adding them
  • Test your presentation on different devices
  • Save frequently to avoid losing work
  • Use the thumbnail view for better overview

📱 Mobile Friendly

  • The app works great on tablets and phones
  • Swipe gestures work in presentation mode
  • Buttons adapt to touch interfaces
  • Text scales appropriately on small screens
